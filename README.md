# TraceGA

全链路埋点监控平台

## 项目结构

```
TraceGA/
│
├── packages/
│   ├── trace-sdk/            # 客户端埋点 SDK
|   |
│   ├── trace-server/         # 后端服务
│   │
│   └── trace-admin/          # 管理平台
│
├── package.json              # monorepo 根配置
├── pnpm-workspace.yaml       # workspace 配置
└── README.md
```

## 开发指南

### 环境要求

- Node.js
